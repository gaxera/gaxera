# Security Policy

## Current Status

Gaxera is pre-v0.1 and **not yet suitable for any real workload**.

## Reporting Vulnerabilities

Found a vulnerability anyway? Please use GitHub's
**[Report a vulnerability](https://github.com/gaxera/gaxera/security/advisories/new)** feature
to open a private security advisory. Do not file a public issue.

## Security Design

Gaxera's architecture is built on capability-based security from the ground up.
See [docs/spec/technical_spec.md](docs/spec/technical_spec.md) for the
security model.
